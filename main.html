Time, Space, Complexity

Time complexity is a way to express how the runtime of an algorithm grows with input size. When evaluating algorithms we describe their Time complexity using asymptotic notation to compare approaches and pick the most efficient strategy; an efficient algorithm performs fewer steps as the input scales and usually leads to faster, more predictable execution.
